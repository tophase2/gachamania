## サイト概要
### サイトテーマ
<欲しいガチャが置いてある店舗を検索できたり、回したガチャを投稿したりできるSNS>

### テーマを選んだ理由
<最近のガチャのクオリティが高く、自身でもマイブームになっている。
欲しい物がどこに置いてあるか検索できると便利だと思うし、
回したガチャを皆で共有できる場があると楽しいのではないかと考え、このテーマにしました。>

### ターゲットユーザ
<ガチャが好きな人>

### 主な利用シーン
<欲しいガチャが置いてある店舗を検索する時、自分が回したガチャを投稿する時>
